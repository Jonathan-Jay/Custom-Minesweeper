A minesweeper template that allows for multiple bomb types, dynamic flags (with real time toggle), and some other useful template functions.
Also bombs are edited in scriptable objects in the inspector, how fun!

The demo also has a customizer menu with some options to share presets too

inspired by https://hempuli.itch.io/cavern-sweeper and https://danielben.itch.io/dragonsweeper

Github pages build can be found here: https://jonathan-jay.github.io/Custom-Minesweeper/
